JavaScript Strings

The escape sequence \\ inserts a backslash in a string.

Six other escape sequences are valid in JavaScript:

'\b' Backspace
'\f' Form Feed
'\n' New Line
'\r' Carriage Return
'\t' Horizontal Tabulator
'\v' Vertical Tabulator
