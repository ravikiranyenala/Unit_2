DHTML means Dynamic HyperText Markup Language, HTML and DHTML are different. With the HTML we can develop Static web pages, but with the DHTML we can develop Dynamic web Pages. with the DHTML we can develop animated web pages, and we can develop gaming web pages. DHTML can supports to big browsers like Microsoft. In DHTML user no need to install or download extra plug-in.
